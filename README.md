# Blog-App

## Requirement

- user can post and publish blog content
- user can see post
- authentication system
- user can see their own profile

## table

- post

  - id
  - title
  - content
  - authorId
  - published
  - publishAt
  - createdAt
  - updatedAt

- user

  - id
  - name
  - email
  - password
  - profile
  - createdAt
  - updatedAt

- profile
  - id
  - bio
  - createdAt
  - updatedAt
  - userId

## technology

- graphql
- typeScript
- postgresql
- prisma
